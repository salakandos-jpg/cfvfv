# World Host — Fabric Mod для Minecraft 1.20.1

Открывай свой одиночный мир для других игроков прямо в игре, с полной панелью управления.

## Возможности

| Функция | Описание |
|---|---|
| **Delete → Меню** | Нажми Delete в мире — откроется главное меню |
| **Открыть мир** | Открыть LAN-мир с настройкой MOTD, макс. игроков, режима и читов |
| **Закрыть мир** | Кикнуть всех и закрыть доступ |
| **Панель управления** | Кик, бан, разбан, смена режима, выдача/снятие OP, ЛС, телепорт |
| **Список забаненных** | Просмотр и управление баном |
| **Настройки** | MOTD, макс. игроков, PvP, уведомления, AFK-кик, белый список и др. |
| **Конфиг** | Автосохранение в `config/worldhost.json` |

## Управление

- **Delete** — открыть/закрыть меню World Host (можно сменить в настройках управления)


## Открытие мира

При нажатии «Открыть мир» откроется экран настройки:
- **MOTD** — название сервера (видно другим игрокам в LAN-меню)
- **Макс. игроков** — от 1 до 100
- **Режим игры** — Выживание / Творческий / Приключение / Наблюдатель
- **Читы** — разрешить команды вроде `/gamemode`

> ⚠ Работает только в одиночном мире. Другие игроки в той же сети подключаются через меню «Многопользовательская игра → LAN».

## Панель управления

Открывается у игрока с правами оператора (OP). Показывает список онлайн-игроков в реальном времени.

### Действия над выбранным игроком:
| Кнопка | Действие |
|---|---|
| ⚡ Кик | Отключить с указанием причины |
| ✖ Бан | Заблокировать навсегда (с причиной) |
| ✔ Разбан | Снять блокировку |
| ◀ Режим ▶ | Выбрать режим и нажать «Сменить режим» |
| ★ Выдать OP | Дать права оператора |
| ☆ Снять OP | Убрать права оператора |
| ✉ Отправить | Личное сообщение игроку |
| ⊕ Телепорт | Телепортировать игрока к себе |
| ↻ Обновить | Обновить список игроков |

## Настройки

| Параметр | По умолчанию | Описание |
|---|---|---|
| MOTD | Добро пожаловать в мой мир! | Название сервера |
| Макс. игроков | 20 | Лимит подключений |
| Режим по умолч. | Выживание | Режим для новых игроков |
| PvP | Вкл | Разрешить PvP |
| Уведомл. входа | Вкл | Сообщение в чат при входе |
| Уведомл. выхода | Вкл | Сообщение в чат при выходе |
| Белый список | Выкл | Только разрешённые игроки |
| Сообщ. о бане | Вкл | Уведомлять OP о банах в чате |
| Список игроков | Вкл | Показывать вкладку игроков |
| Авто-кик AFK | Выкл | Кикать за бездействие |
| AFK таймаут | 30 мин | Время до авто-кика |

Конфиг сохраняется в: `.minecraft/config/worldhost.json`

## Установка

1. Установи [Fabric Loader](https://fabricmc.net/use/) для Minecraft 1.20.1
2. Установи [Fabric API](https://modrinth.com/mod/fabric-api)
3. Скопируй файл `worldhost-1.0.0.jar` в папку `.minecraft/mods/`
4. Запусти Minecraft с профилем Fabric 1.20.1

## Сборка из исходников

**Требования:** Java 17+, подключение к интернету (для загрузки зависимостей)

```bash
cd minecraft-mod

# Windows
gradlew.bat build

# Linux / macOS
chmod +x gradlew
./gradlew build
```

Готовый JAR будет в: `build/libs/worldhost-1.0.0.jar`
