# World Host — Fabric Mod для Minecraft 1.20.1

Открывай свой одиночный мир для **любого игрока в интернете** прямо из игры.  
Другие игроки с этим модом нажимают Delete → видят список доступных миров → заходят одним кликом.

---

## Как это работает

```
  [Хост]                   [Relay API сервер]          [Другой игрок]
    │  ── открыть мир ──>  │ регистрирует адрес │  <── список миров ──  │
    │  <── код ABC123 ───  │  + код сессии      │  ──> подключение ──>  │
    │                      │   (heartbeat 30s)  │      напрямую к хосту  │
```

1. Хост нажимает **Delete → 🌐 Открыть для всего мира**
2. Мод определяет внешний IP, открывает LAN-порт, регистрирует мир на relay-сервере
3. Хост получает **код сессии** (например `A3F9C2`) и свой адрес
4. Любой другой игрок с модом нажимает **Delete → 🌐 Войти в мир** — видит список и подключается

> ⚠️ **Требование**: хост должен либо **пробросить порт** (port forwarding) на роутере,  
> либо использовать **[playit.gg](https://playit.gg)** (бесплатный туннель, специально для игровых серверов).

## Открытие мира глобально

**Delete → 🌐 Открыть для всего мира**

Шаги, которые мод выполняет автоматически:
1. Определяет твой внешний IP через `api.ipify.org`
2. Открывает LAN-порт через Minecraft (IntegratedServer)
3. Регистрирует мир на relay-сервере → получает **короткий код** (6 символов)
4. Запускает heartbeat каждые 30 секунд (мир виден в списке пока ты онлайн)

Настройки при открытии:
- **MOTD** — название твоего мира в списке
- **Макс. игроков** — лимит (1–100)
- **Режим игры** — начальный режим для новых игроков
- **Читы** — разрешить `/gamemode` и другие команды

## Вход в мир другого игрока

**Delete → 🌐 Войти в мир (по интернету)**

| Способ | Описание |
|---|---|
| **Список** | Нажми «↻ Обновить» — увидишь все открытые миры. Кликни и «▶ Войти» |
| **По коду** | Введи код (например `A3F9C2`) в поле — мод найдёт мир |
| **Напрямую** | Введи `IP:порт` (например `1.2.3.4:25565`) — прямое подключение |

Информация о каждом мире в списке:
- Хозяин, MOTD, адрес
- Режим игры, количество игроков, код сессии

| ★ Выдать OP | Дать права оператора |
| ☆ Снять OP | Убрать права оператора |
| ✉ Отправить | Личное сообщение |
| ⊕ Телепорт | Телепортировать к себе |
| ↻ Обновить | Обновить список игроков |

## Настройка relay-сервера (обязательно!)

Чтобы список миров работал, укажи URL своего развёрнутого API-сервера в `worldhost.json`:

```json
{
  "relayUrl": "https://ИМЯ_ПРИЛОЖЕНИЯ.replit.app"
}
```

API-сервер из этого проекта **уже содержит** нужные эндпоинты:

| Метод | Путь | Описание |
|---|---|---|
| `POST` | `/api/worldhost/register` | Зарегистрировать мир |
| `POST` | `/api/worldhost/heartbeat/:code` | Продлить сессию |
| `GET`  | `/api/worldhost/list` | Список открытых миров |
| `GET`  | `/api/worldhost/world/:code` | Мир по коду |
| `DELETE` | `/api/worldhost/close/:code` | Закрыть мир |

Сессии автоматически исчезают через **60 секунд** без heartbeat.

## Как подключиться без проброса портов

Используй **[playit.gg](https://playit.gg)** — бесплатный туннель специально для Minecraft:

1. Зарегистрируйся на playit.gg
2. Скачай и запусти агент (`playit-agent`)
3. Создай TCP туннель → получи адрес вида `8.tcp.us-cal-1.playit.gg:12345`
4. Этот адрес вводи в поле «Прямое подключение» или укажи при регистрации

## Установка мода

1. Установи [Fabric Loader](https://fabricmc.net/use/) для Minecraft **1.20.1**
2. Установи [Fabric API](https://modrinth.com/mod/fabric-api) для 1.20.1
3. Скопируй `worldhost-1.0.0.jar` в папку `.minecraft/mods/`
4. Пропиши `relayUrl` в `config/worldhost.json` (адрес твоего API-сервера)
5. Запусти Minecraft с профилем Fabric 1.20.1
